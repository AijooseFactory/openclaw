# Skill Creator Changelog

All notable changes to the skill-creator skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.4.0] - 2026-03-17

### Added
- Session-based improvement integration (references self-improvement-loop skill)
- Improvement Loop section for continuous enhancement
- tests/, evals/, CHANGELOG.md as required directories

### Changed
- Enhanced to define Skill Creator v2.4.0 specification
- Added 6 Explicit Guarantees for "top 1%" skill quality
- Integrated with writing-skills quality standards

## [2.3.0] - 2026-03-17

### Added
- Six Explicit Guarantees (Tool Quality, Evaluation, Approval/Safety, Context Efficiency, MCP Security, Governance)
- Minimum Observability as Core Required section
- Structural requirement: tests/, evals/, CHANGELOG.md now required (not optional)

### Changed
- Version bumped from 2.2.0 to 2.3.0

## [2.2.0] - 2026-03-17

### Added
- Structural requirement documentation for tests/, evals/, CHANGELOG.md

## [2.1.0] - 2026-03-17

### Added
- Security feedback fixes (MCP authentication options, configurable progress interval)
- Version sync mechanism

## [2.0.0] - 2026-03-16

### Added
- Complete restructuring to Skill Creator v2.4.0 specification
- 10 Core Required sections
- 6 Conditional sections
- 5 Advanced sections
- Enhanced frontmatter with 15 fields

### Changed
- Canonical structure replacement with selective content migration
- Preserved: progressive disclosure, packaging workflow, CSO, token-efficiency

## [1.0.0] - Initial

### Added
- Initial skill-creator implementation
- Basic skill anatomy documentation
- Packaging and validation workflow
